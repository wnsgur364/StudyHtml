HTML이란?

하이퍼 텍스트 마크업 랭귀지(Hyper Text Markup Language)
하이퍼 텍스트를 마크업하는 언어(웹에서 웹페이지를 만드는 언어)
하이퍼 텍스트 : 웹사이트에서 링크를 클릭해 다른 문서나 사이트로 이동하는 것
마크업 : 태그(tag)를 이용하여 문서에서의 제목과 본문, 이미지, 링크 등을 표시하는 것
태그(tag) : html에서 사용하는 명령어(꺽쇠괄호로 묶여있는 것 = 태그 = 요소)
