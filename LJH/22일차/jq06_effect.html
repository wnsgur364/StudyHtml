제이쿼리 효과(effect)

.hide() / .show() 메서드

hide버튼을 누르면 사라지고 show버튼을 누르면 나타난다.

.fadeOut() / .fadeIn() 메서드

fade out버튼을 누르면 서서히 사라지고 fade in버튼을 누르면 서서히 나타난다.

.slideUp() / .slideDown() 메서드

slide up버튼을 누르면 접히고 slide down버튼을 누르면 펼쳐진다.

.toggle() 메서드

toggle버튼을 누르면 .show() / .hide() 가 번갈아 실행된다.

.fadeToggle() 메서드

fade toggle버튼을 누르면 .fadeIn() / .fadeOut() 이 번갈아 실행된다.

.slideToggle() 메서드

slide toggle버튼을 누르면 .slideDown() / .slideUp() 이 번갈아 실행된다.

.fadeTo() 메서드

fade to 버튼을 누르면 지정한 투명도로 적용할 수 있다.
사용법은 $("선택자").fadeTo(효과소요시간, 투명도, 콜백함수);
콜백함수는 생략가능

콜백함수

콜백함수가 실행되면서 버튼이 변경된다.
